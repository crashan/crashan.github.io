Given a string, find the length of the longest substring without repeating characters.

Example 1:

Example 2:

Example 3:

  • 题目大意
    给一个string，求最大不重复字串长度

暴力解法

第一个思路是暴力解法，遍历所有的字串，判断是否没有重复字符串，然后维护max_len。时间复杂度O(n^3)，空间复杂度O(k)

优化解法

第二个思路是，维护一个区间[i, j], 遍历字符串，判断当前字符是否在区间内，是的话i+1，否则j+1，从0开始遍历，保证每次都可以复用上一次的区间set。最优情况下每次执行都是j+1，只需要执行n次，最坏情况下比如aaaaa，每次遍历都是先j+1然后又i+1，需要执行2n次，总的时间复杂度为O(n)，空间复杂度O(n)

优化解法2

在上面的解法中，每次发现有重复字符，都会从前面开始一个一个删除，实际上一次性把所有重复的删除了即可，如果采用map<char, int>，那在删除的前面全部重复字符的时候相当于间接遍历了map，和一个一个删除的方案没有不同。
实际上，可以在map存第一个不同字符的下标，这样每次直接通过索引就可以快速跳区间。

You are given two non-empty linked lists representing two non-negative integers. The digits are stored in reverse order and each of their nodes contain a single digit. Add the two numbers and return it as a linked list.

You may assume the two numbers do not contain any leading zero, except the number 0 itself.

Example:

  • 题目大意
    给两个链表，分别表示两个数字的每一位逆序，求这两个数的和
  • 思路
    直接遍历链表，注意进位即可。时间复杂度O(m + n), 空间复杂度O(1)

Posted on 2019-03-24 | In leet_code , problemset

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

题目大意：给一个数组和一个目标值，数组中有两个数和为目标值，输出两个数的坐标。

  • 思路
    遍历一遍数组，每次都将当前值及其下标插入哈希表，插入前查找map是否存在target-num[i]，是则说明找到。
  • 复杂度
    最坏情况下遍历到最后一个数才找到答案，每次遍历都要查找一次、插入一次，时间复杂度为O(n)，空间复杂度O(n)
  • AC代码
